# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by opening an issue with limited details and requesting a private contact channel.

Please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fix

Do not post sensitive exploit details publicly until the issue is reviewed and addressed.

## Supported Versions

This project is maintained for personal use. Security fixes are provided on a best-effort basis only.
